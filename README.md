# Online Complaint System

## Project Description

The **Online Complaint Registration and Management System** is a full-stack web application developed using the **MERN Stack (MongoDB, Express.js, React.js, and Node.js)**. The system provides a simple and secure platform for users to register complaints, monitor their status, and communicate with administrators throughout the complaint resolution process.

The application is designed to replace traditional paper-based complaint handling with a digital solution that improves efficiency, transparency, and response time. Users can submit complaints under different categories, track progress in real time, and receive status updates. Administrators can view, manage, assign, update, and resolve complaints through a dedicated dashboard.

The system implements secure user authentication and authorization to ensure that only authorized users can access specific features. Complaint data is stored in MongoDB, while the Express.js and Node.js backend provides RESTful APIs that connect seamlessly with the React frontend.

## Key Features

* User registration and secure login
* JWT-based authentication and authorization
* Submit complaints with category and description
* View and track complaint status
* Edit or delete complaints (before processing, if permitted)
* Administrator dashboard for complaint management
* Update complaint status (Pending, In Progress, Resolved, Rejected)
* Search and filter complaints
* Responsive and user-friendly interface
* RESTful API architecture
* MongoDB database integration

## Objective

The main objective of this project is to provide an efficient, transparent, and secure complaint management platform that simplifies complaint registration and resolution while improving communication between users and administrators.

This project also demonstrates the implementation of full-stack web development concepts, including frontend development, backend API creation, database management, authentication, and CRUD operations using the MERN stack.
